# Fork Practice - Suyash

Welcome! This is a practice repository for learning Git forking and branching workflow.

## Purpose

This repository is designed to help beginners practice:
- Forking repositories
- Creating branches
- Making commits
- Pushing changes
- Creating Pull Requests

## How to Contribute

Please read [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions on how to practice the Git workflow.

## What You'll Learn

1. How to fork a repository
2. How to clone your fork locally
3. How to create and switch branches
4. How to make commits
5. How to push branches to GitHub
6. How to create Pull Requests

## Getting Started

1. Fork this repository
2. Follow the instructions in [CONTRIBUTING.md](CONTRIBUTING.md)
3. Add your name to [contributors.md](contributors.md)
4. Submit a Pull Request!
