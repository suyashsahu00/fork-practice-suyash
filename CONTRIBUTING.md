# How to Contribute - Practice Repository

This repository is for practicing Git forking and branching workflow!

## Steps to Follow:

1. **Fork this repository**
   - Click the "Fork" button in the top right corner
   - Create a fork under your account

2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR-USERNAME/fork-practice-suyash.git
   cd fork-practice-suyash

   ```

3. **Create a new branch**
   ```bash
   git checkout -b add-your-name
   ```

4. **Add your name to contributors.md**
   - Open `contributors.md`
   - Add your name and a 2026 goal following the format

5. **Commit your changes**
   ```bash
   git add contributors.md
   git commit -m "Add: Added my name to contributors"
   ```

6. **Push to your fork**
   ```bash
   git push origin add-your-name
   ```

7. **Create a Pull Request**

   - Submit your PR!
